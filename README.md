Scripts to control a fully motorized Nikon Ti equipped with a Hamamatsu ORCA Flash v3 camera.

Scripting based on Micro-Manager and Python (pymmcore_plus).

Motorized (script-controlled) components:
- Nikon Ti (nosepiece, filter turret, halogen lamp)
- Sutter shutter
- Hamamatsu ORCA Flash v3
- Lumencor SOLA laser
- Ludl M6000 Stage Controller
